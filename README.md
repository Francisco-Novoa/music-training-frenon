
## API-TASK

API que permite gestionar tareas utilizando mongoose, expressjs y MongoDbx.

![api-task](trabajo.png)

## RUTAS A CREAR

TASK:  http://localhost:5000/api/task

- GET: 
    `/task`
    `/task/id`

- POST:
    `/task`

- PUT:
    `/task/id`

- DELETE:
    `/task/id`



USER:  http://localhost:5000/api/user

- GET: 
    `/user`
    `/user/id`

- POST:
    `/user`

- PUT:
    `/user/id`

- DELETE:
    `/user/id`
